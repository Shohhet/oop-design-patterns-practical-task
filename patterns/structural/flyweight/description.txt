Предположим у нас есть автотранспортное предприятие, которое занимается перевозкой различных типов грузов
для которых нужны специализированные автомобили. Например бетон, строительная техника, лес, уголь.
Соответственно для перевозки таких грузов нужны бетоновоз, трал, лесовоз, самосвал.
Но покупать все машины сразу слишком дорого, постоянно нанимать новую машину тоже,
поэтому машину покупают при наличии заявки.
Соответственно для всех последующих заявок этого типа уже используется машина из гаража.
